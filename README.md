# Employee Payroll Management System

## Technologies Used

- Core Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript
- JPA/Hibernate

## Features

- Add Employee
- Update Employee
- Delete Employee
- Search Employee
- Generate Payslip
- Dashboard
- Payroll Calculation

## Run Project

1. Create payroll_db in MySQL
2. Update application.properties
3. Run EmployeePayrollWebApplication
4. Open http://localhost:8081
